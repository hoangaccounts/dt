# Chat End Checklist

## Purpose

This checklist ensures durable knowledge is captured and context stays accurate at the end of a productive session.

---

## End of Session Checklist

- [ ] Run `@summary` per chat-annotation-protocol
- [ ] Review and confirm notes
- [ ] Review and confirm updates (per file)
- [ ] Apply updates only with baseline files
- [ ] Persist context via `dt ai-context update` (optional)
- [ ] Confirm session closure

---

*Silence never mutates state.*
